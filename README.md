# 🐉 Dragon Runner Game

> A fun dragon runner game where you dodge obstacles to survive as long as possible — built with pure JavaScript, HTML & CSS

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=white) ![HTML](https://img.shields.io/badge/HTML-555555?style=for-the-badge&logo=html&logoColor=white) ![CSS](https://img.shields.io/badge/CSS-555555?style=for-the-badge&logo=css&logoColor=white)

## 📖 About

A fun dragon runner game where you dodge obstacles to survive as long as possible — built with pure JavaScript, HTML & CSS

## ✨ Features

- Smooth dragon running animation
- Randomly generated obstacles
- Score counter & high score
- Space bar / tap to jump
- Game over & restart screen

## 🛠️ Tech Stack

- **JavaScript**
- **HTML**
- **CSS**

## 🚀 Getting Started

```bash
git clone https://github.com/Paras1Chauhan/dragon-runner-game.git
cd dragon-runner-game
open index.html in your browser
```
